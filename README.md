## 📖 캡스톤 디자인1 프로젝트
---
🧳 여행 계획을 돕는 앱, travelmaker<br>
- 의도 분류 모델을 이용해 **챗봇** 구현 및 API 통신
- 한국관광공사 API를 이용해 각 지역별 정보를 시각화
- **파이어베이스**에 사용자 데이터 저장

#### 📖 프로젝트 설명
---
1. 주요 기능:&nbsp;사용자가 원하는 활동을 분류하여 관광지나 식당을 추천해주는 챗봇, 사용자끼리 소통할 수 있는 커뮤니티, 전국의 관광 정보를 한 눈에 볼 수 있는 지역 정보 등
